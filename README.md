## Cloudflare Page 部署

>[!NOTE]
>需要 Cloudflare 账户

1. 首先 fork 该仓库
2. 然后登陆 Cloudflare，在首页打开【Workers & Pages】>> 二级菜单的【Overview】>> 点击主页的【Create application】按钮 >> 点击【Pages】>> 最后点击【Connect to Git】即可
